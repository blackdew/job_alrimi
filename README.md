# Job Alrimi (남해군 일자리·빈집 정보 알림 앱)

> 남해군 주민 및 예비 이주민을 위한 지역 밀착형 생활 정보 플랫폼

## 프로젝트 개요

복잡한 검색 없이 **"알림(Push) 받고 → 전화(Call) 거는"** 직관적 프로세스를 제공하는 모바일 앱입니다.

### 해결하고자 하는 문제
- **Web의 한계**: 기존 지자체 홈페이지 기반 정보 제공의 모바일 환경 부적합
- **정보 비대칭성**: 높은 탐색 비용으로 주민들이 양질의 일자리/정책 혜택을 놓침
- **이주 장벽**: 외부 예비 이주민들이 현지 주거(빈집) 정보를 얻기 어려움

## 수집 대상 데이터

### 일자리 정보
1. 남해군청 유관기관 소식 (새올)
2. 남해군청 구인구직 게시판
3. 경남 워크넷 (남해군)

### 빈집 정보
1. 남해군 귀농귀촌지원센터 (`refarm.namhae.go.kr`)
2. 그린대로 농촌 빈집은행 (`greendaero.go.kr`)

## 타겟 사용자

| 세대 | Pain Point | Solution |
|------|-----------|----------|
| 3040 | 생업으로 바빠 홈페이지 확인 어려움 | 실시간 푸시 알림으로 경쟁 우위 선점 |
| 50대 | 정보 욕구 높으나 신체적 변화 | 정보의 질 + 눈이 편안한 UI |
| 6070 | 복잡한 UI/작은 글씨 장벽 | Push to Call - 알림→전화 2단계 |
| 이주민 | 현지 방문 없이 정보 획득 불가 | Work & Stay 통합 알림 |

## 핵심 기능

1. **실시간 푸시 알림**: 사용자 설정 키워드 기반 즉시 알림
2. **Push to Call UI**: 화면 중앙 대형 '전화 걸기' 버튼
3. **키워드 구독**: 관심 분야(밭일, 사무직, 월세, 빈집 매매) 체크박스 설정

## 기술 스택

### Backend (크롤링)
- **Runtime**: Node.js
- **Scraping**: Playwright (Headless Browser)
- **Parsing**: cheerio

### Database
- **Firebase Firestore**: NoSQL, 실시간 동기화

### Frontend
- **Framework**: Flutter (Android/iOS/Web 동시 대응)
- **Push**: Firebase Cloud Messaging (FCM)

## 데이터 처리 프로세스

```
수집(Collect) → 필터링(Dedup) → 가공(Parse) → 저장(Store) → 트리거(Trigger)
```

1. 주기적(30분 간격) 게시판 스캔
2. 신규 게시물 식별 (중복 제거)
3. 전화번호 추출 및 키워드 태깅
4. Firestore 저장
5. Cloud Functions로 푸시 알림 발송

## 로드맵

### Phase 1: MVP
- 일자리/빈집 알림
- "알림 → 전화" 핵심 기능

### Phase 2: 고도화
- 사용자 피드백 반영
- 상세 필터링 (거리순, 세부 직종)

### Phase 3: 플랫폼 확장
- 생활 밀착 정보 (농기계, 장날, 보건소)
- "지역 생활 정보 허브"
